# マイおうち

自分だけの家を作るカスタマイザー。SVG でリアルタイムプレビュー。

- **屋根**: 三角／平／丸／段々 × 6色
- **壁**: 無地／木板／レンガ／しま × 6色
- **ドア**: 木／モダン／アーチ／ガラス × 6色
- **窓**: 四角／丸／アーチ／格子
- **庭**: 木／花壇／ポスト／街灯／柵／ベンチ／噴水／ねこ（タップで切替）
- **風景**: 昼／夕方／夜（窓に明かり）／雪
- 🎲 ランダムボタンで全部おまかせ
- LocalStorage 保存
- Xシェア
